Ukraine Geospatial Visualization
Виконав: Стоян Олександр

Цей проєкт реалізує веб-сервер та Python-інструменти для відображення геопросторових даних України. Він використовує Flask для серверної частини, PostgreSQL з розширенням PostGIS для збереження геоданих та Leaflet для інтерактивної візуалізації на мапі. Основні функції включають:
	Завантаження кордону України з відкритих джерел
	Генерація квадратної сітки з заданим розміром (10 км)
	Обрізання сітки по кордону України
	Генерація секторів із заданими азимутами (0°, 120°, 240°)
	Відображення даних на карті за допомогою Leaflet

Функціональність
Імпорт та обробка геопросторових даних:
	Завантаження та збереження кордону України.
	Генерація сітки та секторів.
	Збереження даних в базі даних PostgreSQL/PostGIS:
	Створення таблиць для збереження даних.
	Обрізання геометрій по кордону України.

Веб-візуалізація:
	Сервер на Flask для надання API з доступом до геоданих.
	Відображення інтерактивної карти на основі Leaflet.
Вимоги:
	Python 3.x
	PostgreSQL з розширенням PostGIS
	Flask, SQLAlchemy, GeoPandas


Завдання:
1. Знайти координати кордону України та занести їх у базу даних Oracle/PostgreSQL/MySQL.

2. Графічно відобразити кордон у середовищі Python або на веб-інтерфейсі за допомогою бібліотеки Leaflet.

3. Запропонувати алгоритм, який розбиватиме карту України на однакові квадрати (сторона ~1 км, можна і більше, якщо не вистачає ресурсів на ноутбуці).

4. Зберегти вершини сформованих квадратів у базу даних Oracle/PostgreSQL/MySQL.

5. Графічно відобразити ці квадрати у середовищі Python на карті або на веб-інтерфейсі за допомогою бібліотеки Leaflet.

6. У вас є збережені координати вершин квадратів. З кожної вершини графічно зобразити 3 сектори з азимутами 0, 120 та 240 градусів, розкривши їх на 60 градусів радіусом 5 км. Запропонувати алгоритм, який обраховуватиме, які вершини сформованих квадратів перетинає кожен сектор. Результати перетину зберегти в БД.

7. Графічно відобразити ці сектори поверх квадратів у середовищі Python на карті або на веб-інтерфейсі за допомогою бібліотеки Leaflet.


Використані бібліотеки
Для реалізації через Python:
	geopandas: робота з геопросторовими даними.
	shapely: створення геометричних фігур (полігонів, секторів).
	numpy: математичні операції, обчислення координат.
	matplotlib: візуалізація карти та графіків.
	sqlalchemy: робота з базою даних через ORM.
	psycopg2: підключення до PostgreSQL.
	multiprocessing: розпаралелювання завдань для генерації секторів.
Для реалізації через Flask (веб-сервер):
	flask: запуск веб-сервера та обробка HTTP-запитів.
	flask_cors: підтримка CORS для доступу до API.
	leaflet.js: візуалізація карти на веб-інтерфейсі.
	geoalchemy2: робота з геоданими у PostgreSQL через SQLAlchemy.
База даних:
	PostgreSQL
	PostGIS: для роботи з геоданими

Коротка інструкція з запуску програми:
Настройка бд:
	1.Встановіть PostgreSQL з підтримкою PostGIS.
	2.Створіть базу даних (geoproject)
	3.Увімкнути розширення PostGIS
	4.Надати прав користувачу на редагування БД 

Python:
	1. Змінити дані для підключення до бд в main.py
	2. Встановити бібліотеки які написані вище.
	3. Запустіть main.py (таблиці будуть створені та заповнені автоматично).
	4. Для відображення через веб запустіть server.py змінивши дані підключення до БД. Зверніть увагу: запуск можливий лише після того, як у БД вже створено таблиці та завантажено необхідні дані. Це означає, що перед запуском веб-сервера потрібно виконати main.py.
